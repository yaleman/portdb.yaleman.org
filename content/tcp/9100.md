## IANA Data

_Name:_ pdl-datastream

_Description:_ Printer PDL Data Stream

_Note:_ The protocol name "pdl-datastream" is primarily registered for use
in DNS SRV records (RFC 2782). DNS SRV records allow a protocol to run on
any port number, but the default port for this protocol is 9100.
